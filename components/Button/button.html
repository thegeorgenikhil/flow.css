Buttons

Use flow.css custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

You are viewing this on a smaller screen(less than 600px) so all the button styles are by default, converted to display : block

Contained Buttons

Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app.

btn bg-[variant] color-[variant](OPTIONAL) 

                      
        <button type="button" class="btn bg-primary">Primary</button>

        <button type="button" class="btn bg-secondary">Secondary</button>

        <button type="button" class="btn bg-success">Success</button>

        <button type="button" class="btn bg-danger">Danger</button>

        <button type="button" class="btn bg-warning color-dark">Warning</button>

        <button type="button" class="btn bg-info">Info</button>

        <button type="button" class="btn bg-dark">Dark</button>

        <button type="button" class="btn btn-disabled" disabled>Disabled</button>

        <button type="button" class="btn bg-white color-dark">Ghost</button>

        <button type="button" class="btn btn-link">Link</button>

Outline Buttons

In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the btn-outline-* ones to remove all background images and colors on any button.

btn btn-outline-[variant] 

                      
        <button type="button" class="btn btn-outline btn-outline-primary">Primary</button>

        <button type="button" class="btn btn-outline btn-outline-secondary">Secondary</button>

        <button type="button" class="btn btn-outline btn-outline-success">Success</button>

        <button type="button" class="btn btn-outline btn-outline-danger">Danger</button>

        <button type="button" class="btn btn-outline btn-outline-warning">Warning</button>

        <button type="button" class="btn btn-outline btn-outline-info">Info</button>

        <button type="button" class="btn btn-outline btn-outline-dark">Dark</button>

Button Sizes

Button Sizes can be changed by using the text-size utility classes.

btn bg-[variant] text-[variant] 

                      
        <button type="button" class="btn bg-primary text-lg">Primary</button>
        <button type="button" class="btn bg-primary text-md">Primary</button>
        <button type="button" class="btn bg-primary text-xs">Primary</button>

Buttons with Icon

Icon Buttons can be created, by using a SVG as a child to parent button.

btn bg-[variant] color-[variant](OPTIONAL) 

                      
        <button type="button" class="btn btn-block bg-primary">
          <i class="fas fa-upload"></i> Upload
        </button>
        <button type="button" class="btn bg-primary">
          <i class="fas fa-upload"></i> Upload
        </button>

Floating Action Buttons

A floating action button (FAB) performs the primary, or most common, action on a screen. It appears in front of all screen content, typically as a circular shape with an icon in its center.

btn btn-floating bg-[variant] color-[variant](OPTIONAL) 

                      
    <button type="button" class="btn btn-floating bg-primary">
      <i class="fas fa-upload"></i>
    </button>
    <button type="button" class="btn btn-floating bg-danger">
      <i class="fas fa-save"></i>
    <button type="button" class="btn btn-floating bg-success">
      <i class="fas fa-search"></i>
    </button>
