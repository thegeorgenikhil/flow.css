Alerts

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

Note: By default, the alert component takes the width of the parent element

Simple Alerts

Simple alerts can used by adding alert bg-[variant] class eg: alert bg-primary

This is a primary alert—check it out!
This is a secondary alert—check it out!
This is a success alert—check it out!
This is a danger alert—check it out!
This is a warning alert—check it out!
This is a info alert—check it out! 
                      
        <div class="alert bg-primary">This is a <strong>primary</strong> alert—check it out!</div>

        <div class="alert bg-secondary">This is a <strong>secondary</strong> alert—check it out!</div>

        <div class="alert bg-success">This is a <strong>success</strong> alert—check it out!</div>

        <div class="alert bg-danger">This is a <strong>danger</strong> alert—check it out!</div>

        <div class="alert bg-warning">This is a <strong>warning</strong> alert—check it out!</div>

        <div class="alert bg-info">This is a <strong>info</strong> alert—check it out!</div>

Outline Alerts

Alert Outlines can used by adding alert alert-outline-[variant] color-[variant] class eg: alert alert-outline-primary color-primary bg-primary

This is a primary alert—check it out!
This is a secondary alert—check it out!
This is a success alert—check it out!
This is a danger alert—check it out!
This is a warning alert—check it out!
This is a info alert—check it out! 
                      
        <div class="alert alert-outline-primary color-primary">This is a <strong>primary</strong> alert—check it out!</div>

        <div class="alert alert-outline-secondary color-secondary">This is a <strong>secondary</strong> alert—check it out!</div>

        <div class="alert alert-outline-success color-success">This is a <strong>success</strong> alert—check it out!</div>

        <div class="alert alert-outline-danger color-danger">This is a <strong>danger</strong> alert—check it out!</div>

        <div class="alert alert-outline-warning color-warning">This is a <strong>warning</strong> alert—check it out!</div>

        <div class="alert alert-outline-info color-info">This is a <strong>info</strong> alert—check it out!</div>

Alert With Icons

Icons can be added easily by simple putting them with the alert content

This is a primary alert—check it out!
This is a primary alert—check it out! 
                      
        <div class="alert bg-primary">
          <i class="fas fa-check-circle"></i> This is a <strong>primary</strong> alert—check it out!
        </div>
        <div class="alert alert-outline-primary color-primary">
          <i class="fas fa-check-circle"></i> This is a <strong>primary</strong> alert—check it out!
        </div>

Dismissible Alerts

User can dismiss the alerts by clicking on the close icon

This is a primary alert—check it out!
This is a primary alert—check it out!
                      
        <div class="alert alert-dismissible bg-primary">
            <div class="alert-content">This is a <strong>primary</strong> alert—check it out!</div>
            <button class="alert-dismiss"><i class="fas fa-times"></i></button>
          </div>
          <div class="alert alert-outline-primary alert-dismissible color-primary">
            <div class="alert-content">This is a <strong>primary</strong> alert—check it out!</div>
            <button class="alert-dismiss"><i class="fas fa-times"></i></button>
          </div>
