Input

Easily create forms by using our pre-made input components, along with validation and error styles

Input

Input components takes up the width of the entire parent component.Label and input component is enclosed in the .input-group 

                      
        <div class="input-group">
            <label for="username">Username</label>
            <input type="text" class="input" placeholder="Enter Username:" name="username"/>
          </div>

          <div class="input-group">
            <label for="ph-number">Phone Number</label>
            <input type="number" class="input" placeholder="Enter Phone Number: " name="ph-number"/>
          </div>

          <div class="input-group">
            <label for="email">Email</label>
            <input type="email" class="input" placeholder="Enter Email: " name="email"/>
          </div>
          
          <div class="input-group">
            <label for="password">Password</label>
            <input type="password" class="input" placeholder="Enter Password: " name="password"/>
        </div>

        <div class="input-group">
            <label for="password">Text Area</label>
            <textarea name="textarea" placeholder="Enter Text: " id="" class="input" cols="30" rows="10"></textarea>
        </div>

Disabled Input

Disable an input component by using .input-disabled 

                      
        <div class="input-group">
            <label for="email">Email</label>            
            <input type="email" placeholder="Email" name="email" value="value" class="input input-disabled" disabled />
        </div>

Input Validations

Validate input from the user by changing the border color using the .is-error and .is-success class.

Not a valid email format!

Good to go!

                      
        <div class="input-group">
            <label for="email">Email</label>
            <input type="email" placeholder="Enter Email: " name="email" class="input is-error" />
            <p class="validation-message color-danger">Not a valid email format!</p>
        </div>
        <div class="input-group">
            <label for="email">Email</label>
            <input type="email" placeholder="Enter Email: " name="email" class="input is-success"/>
            <p class="validation-message color-success">Good to go!</p>
        </div>
