Avatar

Avatars are typically used to display a user profile as a picture, an icon, or short text. Chose from different shapes and sizes

Basic Avatars

Avatars can be created using the avatar class. 

                      
        <img src="url-of-image" alt="avatar" class="avatar"/>

Avatar Sizes

Change the avatar sizes using avatar-[lg | md | sm] along with the avatar class. 

                      
        <img src="url-of-image" alt="avatar" class="avatar avatar-lg"/>
        <img src="url-of-image" alt="avatar" class="avatar avatar-md"/>
        <img src="url-of-image" alt="avatar" class="avatar avatar-sm"/>

Squared Avatars

You can make the avatars as square by using avatar-square . 

                      
        <img src="url-of-image" alt="avatar" class="avatar avatar-lg avatar-square"/>
        <img src="url-of-image" alt="avatar" class="avatar avatar-md avatar-square"/>
        <img src="url-of-image" alt="avatar" class="avatar avatar-sm avatar-square"/>

Avatar With Icons

Icons can also be used as avatars. 

                      
        <div class="avatar-icon bg-primary"><i class="fab fa-github"></i></div>
        <div class="avatar-icon bg-secondary"><i class="fab fa-github"></i></div>
        <div class="avatar-icon bg-success"><i class="fab fa-github"></i></div>
        <div class="avatar-icon bg-danger"><i class="fab fa-github"></i></div>
        <div class="avatar-icon bg-warning"><i class="fab fa-github"></i></div>
        <div class="avatar-icon bg-info"><i class="fab fa-github"></i></div>
        <div class="avatar-icon avatar-square bg-dark"><i class="fab fa-github"></i></div>

Avatar With Text
