Typography

Documentation and examples for flow.css typography, including global settings, font-sizes, family, weights, colors, and more.

Font Family

flow.css uses Inter, as its default font, which is a free font available at Google Fonts. 

                    
        /* By Default imported in flow.css */
        @import url("https://fonts.googleapis.com/css2?family=Inter:wght@200;300;400;500;600;700;800;900&display=swap");

Font Sizes

flow.css uses class based font sizes instead of using the heading tags. We have decided to uses classes as it provides more flexibility to the user. Apart from that all the heading tags, are available and you can even use a mixture of heading tags and font-size classes for better accessibility. For classes you can chose from 7 sizes:

This is extra-large text

This is large text

This is medium text

This is regular text

This is base text

This is small text

This is extra-small text

                    
        <p class="text-xl">This is extra-large text</p>

        <p class="text-lg">This is large text</p>

        <p class="text-md">This is medium text</p>

        <p class="text-regular">This is regular text</p>

        <p class="text-base">This is base text</p>

        <p class="text-sm">This is small text</p>

        <p class="text-xs">This is extra-small text</p>
