Customize

There are times when you need something different than what's available. Learn how to customize flow.css to suit your needs.

Writing Custom Classes

The easiest way to customize is to make your own stylesheet and by linking it in your html file. By this way you can write custom properties and styles and then you just need to add those classes to your elements or components.

In order to override base classes, users can add their own class to the pre-existing classes, and style them seperately, as shown below. 

                      
        <button class="btn btn-primary custom-style">Primary Button</button>
                        
        <-- Custom class styling -->
        <style>
            .custom-style{
                color : black;
                background-color : pink; 
            }
        </style>
