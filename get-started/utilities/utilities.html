Utility Classes

For faster mobile-friendly and responsive development, flow.css includes dozens of utility classes for background-color, border radii, etc.

Background Color

Apply different background colors using .bg-[variant] class.

Class Name Background Color
.bg-primary
#0d6efc
.bg-secondary
#012e41
.bg-success
#10b956
.bg-danger
#dc3545
.bg-warning

#0dcaf0
.bg-dark
#121212

Border Radius

Apply different border-radius using .rounded-[variant] class.

Class Name Border Radius Properties
.rounded-xs 3px
.rounded-sm 5px
.rounded-md 10px
.rounded-lg 15px
.rounded-full 999px

Box Shadow

Apply box-shadow to elements using the .shadow-[variant] class.

Class Name Box Shadow
.shadow-sm
.shadow-base
.shadow-md
.shadow-lg
.shadow-xl

Text Align

Align texts using the .text-[variant] class.

Class Name Text Alignment
.text-left
text-align:left;
.text-center;
text-align:center
.text-right
text-align-right;

Line Divider

Make a line divider using the .divider-sm class.

By default the divider takes up 100% width of the parent element.
